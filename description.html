Game Logic:

The HTML file is like a template of an application. All buttons, text labels, titles and etc. are specified there. The CSS file is used to define the styles of the webpage. The logic of the game is described in the JS file.

When the start button is pressed canvas element appears. setInterval() method calls three different functions:

I) addRects() - spawn number of new blocks in one second;

II) showTime() - update time value every second;

III) loop() - updates the screen every 25 milliseconds.

When the player loses, the restart menu appears and setInterval() methods stop calling the functions.

The player can choose the number of blocks generated per second. 'list' array is used to store block objects.

The player can use right and left keys to move the paddle.
